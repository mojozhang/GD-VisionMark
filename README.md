# GD-VisionMark
基于 OpenCV 的工业橡胶管表面标线长度视觉检测系统（Proof of Concept）

## 📌 项目简介
本项目旨在通过计算机视觉方法，自动检测并精准测量深色/黑色橡胶管表面的特定颜色（如黄色/绿色）划线的物理长度。
专为恶劣光照及多颜色干扰的工业现场环境设计，采用“一键取色包容算法”、“交互式 ROI 抗干扰遮罩”以及“最小外接旋转矩形测长抗锯齿”技术，有效克服传统边缘计算在粗糙表面反光所带来的误差膨胀问题。

## 🎯 核心特性
- **极简双点物理标定**：无需打印复杂的黑白棋盘格，在软件画面直接使用鼠标提取直观的两点进行欧氏距离与物理距离的桥接换算。
- **一键精准色彩截取**：工业现场各种标线难免有磨损、泛白或色散。在画面上单击左键，系统自动提取周围颜色并施加广域包容率（下放 S/V 容差阈值），智能抵抗高亮反光造成的失真断裂。
- **自定义 ROI 遮罩限定**：在画面中拖拉出检测区（ROI）。算法采用掩膜（Mask）运算，彻底阻断测量区外其他同色系物品或斑点的干扰。
- **抗凹凸边缘测长算法**：抛弃了沿不规则周长累加的低精度方案，利用 `cv2.minAreaRect` 将目标包裹至最小面积斜矩形，直接提取长边尺寸作为物料划线长度，无视边缘狗牙侵蚀。
- **全屏原生极速 GUI**：放弃带延迟和拖动互斥的 B/S（Web 网页）界面，通过 `PIL.ImageDraw` 与 `cv2.setMouseCallback` 实现毫秒级响应并能原生的输出全中文抗锯齿信息。

## 💻 快速安装

1. **环境准备 (基于 macOS / Linux / Windows 均可)**
建议使用 Python 3.11 及以上版本。

2. **创建配套虚拟环境**
```bash
python3 -m venv venv
source venv/bin/activate  # macOS / Linux
# windows: venv\Scripts\activate
```

3. **安装依赖资源**
```bash
pip install -r requirements.txt
```

## 🚀 操作工作流

### 第一阶段：环境标定 (赋予系统物理标尺感)
每次使用新的摄相头、或者改变被测物体与镜头的距离后，都需要运行此脚本进行一比一标定。
```bash
# 请一定确保环境已经 source 激活
python3 calibrate_camera.py
```
**操作方法：**
1. 弹出画面后，用鼠标在画面上**左键单击**物体的一端（A 点）。
2. 在物体的另一端再**左键单击**一次（B 点）。*(点错可按右键重置)*
3. 回到终端命令行中，输入该 A、B 线段在现实世界里的 **毫米(mm)** 长度。
4. 按回车，系统将生成 `./camera_params.npz` 并自动存入空间缩放比例文件。

### 第二阶段：精准测长监控 (日常运行)
取得转换系数参数后进入正是检测。
```bash
python3 measure_hose_line.py
```
**操作方法：**
1. **智能取色**: 在目标黄/绿线上，单击**鼠标左键**（单点，不拖动）。滑块会自动根据取样色调整至最优阈值容差。
2. **框选剔除背景**: 在目标范围的左上方**按住鼠标左键不放并向右下拖拉**，松开鼠标后将生成一个蓝色限定锁死区（ROI），这代表任何框范围外的黄色干扰都会被判定为无效废点。
3. **取消限定**: 任意位置单击**鼠标右键**恢复全屏捕捉。
4. **人工精修（进阶）**: 左侧弹出的 `Control Panel` 拥有 6 个直观滑块（HSV 色相、饱和度、亮度上下限），可实时拉动观测检测矩形（红色区域）在不同光效下的包络性。
5. 按 `q` 键立即退出所有挂载。

## 🛠️ 后续工业化重构与硬件升级规划
由于普通 USB 摄像头的“近大远小”透视效应及镜头内边缘严重模糊的缺陷，在精密毫米级计量任务中往往存在物理测量上限壁垒（如因为粗糙边缘晕染或高度摆放差异带来的 ±3mm 到 ±5mm 漂移）。
针对上述瓶颈，本项目已从软件结构上做好了兼容与抽离准备。**后续计划将通过软硬件双重升级重构为真正的桌面级微米/毫米视觉质检工站。**

### 1. 硬件换代：从普通相机到“机器视觉主件”
- **物方远心镜头 (Telecentric Lens)**：选购定倍远心镜头替换常规镜头。依靠其独特的光学特性**彻底消灭透视畸变**。从此无论橡胶管略微垫高或放偏，投射到靶面内的像素高度恒等！
- **工业级全局快门相机 (Global Shutter Camera)**：弃用自带低速滚动的 Webcam 以防抓拍拖影，升级为带 GigE (千兆网) / USB3 的机器视觉相机（如海康、大恒、Basler），并改用专门的厂方 SDK 或兼容的 DirectShow 通道提取无损帧像。
- **针对性偏振环形光源**：部署一套可调亮度的机器视觉光源。通过物理手段过滤反光面并照亮有色划线，使软件的色彩提取对比度逼近 100%，不再畏惧车间复杂暗光。

### 2. 软件进化：融入流水线与工业通讯网络
- **固定式特征提取夹具**：建立标准化的软件待测模板（Template Matching），自动识别来料位置，只有来料落入特定的识别阈架内，才开始执行线长判断逻辑（或交由外部 PLC 触发硬件 IO）。
- **上下游品控打通**：
  - 引进 `Modbus TCP` / `OPC UA` 协议脚本：一旦软件检测出橡胶管线长越界（过长或过短），立即通过工业协议发送 Reject 信号至外接的自动化气缸或者分拣机械臂。
  - 数据持久层重构：每一次检出的结果配同时间戳及判定依据（那张带红框的留存图片），通过 API 推送入库用于后期的产线质量追溯与不良率生成看板。
